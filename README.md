# Cybersecurity Portal

An interactive, responsive web application designed to raise awareness about digital safety, cyber threats, and online fraud prevention. This portal serves as a central hub for users to learn about common cyberattacks, adopt best safety practices, and find resources to report malicious activities.

## 🚀 Features

* **Interactive Navigation:** Clean navbar with links to dedicated sections (Home, Attacks, Safety Tips, Report Fraud, and Resources).
* **Modern UI/UX:** High-contrast, accessible design featuring a clean color palette (Navy Blue and White) for enhanced readability.
* **Call-to-Action (CTA):** Intuitive layout with immediate routing to threat exploration for seamless user engagement.
* **Responsive Layout:** Fully optimized for seamless viewing across desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack

* **HTML5:** Structured semantic markup for a web-accessible layout.
* **CSS3:** Custom styles, Flexbox layout navigation, and smooth transitions.
